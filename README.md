# AI Powered HRMS Assistant

An AI-powered HRMS Assistant built using Google Apps Script, Google Sheets, Claude AI, HTML, CSS, and JavaScript.

## Features
- AI HR Assistant
- Recruitment SOP
- Onboarding Checklist
- Exit Process
- Performance Management
- Email Templates

## Tech Stack
- Google Apps Script
- Google Sheets
- Claude AI API
- HTML
- CSS
- JavaScript
